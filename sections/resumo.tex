\begin{resumo}
O objetivo desse trabalho é estudar as Células de Renshaw e suas funções 
usando modelos computacionais baseados em dados experimentais. A maioria dos dados usados aqui vêm
de experimentos realizados em membros inferiores de gatos. Como esse tipo de
experimento raramente é realizado hoje em dia, esses dados atualmente são
obtidos de ratos. Dessa forma, dados de ratos serão usados com cautela, com 
o devido aviso.
